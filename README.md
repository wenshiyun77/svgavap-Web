# svgavap-Web
一站式社交App物料编辑器的客户端产品介绍和下载页

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 下载地址配置

页面默认使用这些路径：

- `/downloads/svga-vap-lab-mac-arm64.zip`
- `/downloads/svga-vap-lab-mac-x64.zip`
- `/downloads/svga-vap-lab-windows-x64.exe`

也可以通过环境变量覆盖：

- `VITE_DOWNLOAD_MAC_ARM64`
- `VITE_DOWNLOAD_MAC_X64`
- `VITE_DOWNLOAD_WINDOWS_X64`
